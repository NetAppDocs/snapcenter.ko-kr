= 자동으로 검색된 MySQL에 대해 지원되는 복원 전략 유형입니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapCenter를 사용하여 복원 작업을 성공적으로 수행하려면 먼저 전략을 정의해야 합니다.

전체 리소스 복원은 자동으로 검색된 MySQL 데이터베이스에 대해 지원되는 복원 전략입니다. 이렇게 하면 리소스의 모든 볼륨, qtree 및 LUN이 복원됩니다.
