= 클론 복제 워크플로우
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
클론 워크플로우에는 계획, 클론 작업 수행 및 작업 모니터링이 포함됩니다.

다음과 같은 이유로 데이터베이스를 복제할 수 있습니다.

* 응용 프로그램 개발 주기 동안 현재 데이터베이스 구조 및 콘텐츠를 사용하여 구현해야 하는 기능을 테스트합니다.
* 데이터 추출 및 조작 도구를 사용하여 데이터 웨어하우스를 채웁니다.
* 실수로 삭제 또는 변경된 데이터를 복구합니다.
